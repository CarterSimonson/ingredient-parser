Utilities#

ingredient_parser.utils.average(labels, scores, key)#

Average the scores for labels matching key

Given a particular label (key), find the indices of the labels that have that key. Then use those indices to take the average of the associated score in the scores input.

Parameters:

  • labels (List[str]) – List of labels to search key for

  • scores (List[float]) – Confidence score for each label

  • key (str) – Key to calculate confidence for

Returns:

Confidence, average of all labels with given key

Return type:

float

ingredient_parser.utils.find_idx(labels, key)#

Find indices of elements in list matching key.

Return the indices of every element in the input list whose value is equal to the given key. If there is an element with the COMMA label before an element with label key, include this in the list of matched indices

Parameters:

  • labels (List[str]) – List to search for key in

  • key (str) – Key to find in list

Returns:

List of indices of elements with label key Includes elements with label COMMA if the element is previous to a key label

Return type:

List[int]

Examples

>>> find_idx(["QTY", "UNIT", "NAME", "NAME", "COMMENT", "COMMENT"], "NAME")
[2, 3]

>>> find_idx(["QTY", "UNIT", "NAME", "NAME", "COMMA", "COMMENT", "COMMENT"], "COMMENT")
[4, 5, 6]
ingredient_parser.utils.fix_punctuation(sentence)#

Fix punctuation when joining a list into a string

  1. Remove the “, “ from start of the sentence

  2. Remove the space following an opening parens “(” and the space preceeding a closing parens “)” caused by using “ “.join to turn a list into a sentence

  3. Remove the space preceeding a comma

Parameters:

sentence (str) – Sentence in which to fix punctuation

Returns:

Modified sentence

Return type:

str

Examples

>>> fix_punctuation(", some words")
'some words'

>>> fix_punctuation("( text in brackets )")
'(text in brackets)'

>>> fix_punctuation("a comma follows this text ,")
'a comma follows this text,'
ingredient_parser.utils.group_consecutive_idx(idx)#

Yield groups of consecutive indices

Given a list of integers, yield groups of integers where the value of each in a group is adjacent to the previous element’s value.

Parameters:

idx (List[int]) – List of indices

Yields:

List[List[int]] – List of lists, where each sub-list contains consecutive indices

Examples

>>> groups = group_consecutive_idx([0, 1, 2, 4, 5, 6, 8, 9])
>>> [list(g) for g in groups]
[[0, 1, 2], [4, 5, 6], [8, 9]]
ingredient_parser.utils.join_adjacent(tokens, idx)#

Join tokens with adjacent indices in idx list into strings.

Given a list of tokens and list of indices for token with a particular value, join all the token with adjacent indices in the idx list into space seperated strings.

If idx is an empty list, return an empty string.

If there is only one group of adjacent values in idx, return a string.

If there are multiple groups of adjacent values in idx, return a list of strings.

Parameters:

  • tokens (List[str]) – List of ingredient sentence tokens

  • idx (List[int]) – Indices of tokens to group and join

Returns:

List of strings, with adjacent tokens joined If the list only contains one element, return as a string

Return type:

Union(str, List[str])

Examples

>>> join_adjacent(["a", "b", "c", "d", "e", "f"], [0, 1, 3, 4, 5])
['a b', 'd e f']
